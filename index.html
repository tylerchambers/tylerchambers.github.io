Personal

  • Electrum Client - An electrum client library written in Python focused on simplicity and usability.
  • AptLib - A Go library for interacting with APT repositories. Unfinished. Currently able to parse sources.list files and download + parse package indicies. Can currently out perform the main apt implementation included in debian and ubuntu, as it downloads and parses sources concurrently. More gains could be made by making use of Cache-Control, ETag, and Last-Modified headers. I paused work on this as I became busy at my main job, and frustrated with the breadth of APT's functionality. If I were to resume this sort of work it would be re-done in Rust to take advantage of its fast regex, and limit focus to only handling HTTP/HTTPS repositories (most common). It would likely also be useful to have an apt daemon running in the background to keep your sources fresh (autoupdate?), and a CLI to interact with it.
  • Bitcoin REST - A REST API gateway that passes Bitcoin Core RPC calls to a running Bitcoin Core instance and returns its response.
